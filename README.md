# Дмитрий Полуэктов — портфолио

Полностью русская версия сайта-визитки системного администратора с фото, dark-luxury стилем, плавными анимациями и лёгким эффектом курсора.

## Запуск через Docker

```bash
docker compose up --build
```

Открыть:
- http://localhost:3000

## Где менять данные

Основной контент:
- `src/App.jsx`

Стили:
- `src/styles.css`

Фото:
- `public/profile.jpg`

## Запуск без Docker

```bash
npm install
npm run dev
```
